Sobre Mim

Olá, Meu nome é Eduardo Sales, tenho 19 anos e no momento estou cursando Análise e Desenvolvimento de Sistemas no UniCEUB.

Atuo como desenvolvedor de software desenvolvendo sistemas com Python e VBA para extração e manipulação de dados, além de automatizar processos. Além disso, tenho experiência em desenvolvimento web com HTML, CSS e JavaScript, e em construir sites responsivos e otimizados.

Tenho habilidades avançadas com WordPress, criando sites dinâmicos e customizáveis, como um sistema de agendamentos para petshops.

Utilizo Git e GitHub para versionamento de código e possuo muitos projetos onde demonstro minhas capacidades. No momento estou em busca da certificação GitHub Foundations, consolidando minhas práticas em controle de versão e colaboração em projetos.
